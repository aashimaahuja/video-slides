```smalltalk title='Summarize'
1. You can have multiple named exports in a file but only have one default export.

2. You need to import named exports in curly braces whereas
   in case of default exports you can import without curly braces

3. Lastly you can import default exported value with any alias.
   However with named exports you have to use the same name or use as keyword
   if you want to use another name
```
